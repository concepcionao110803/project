## Project Overview

This final project is a **personal portfolio website** designed to showcase my professional biography, technical skills, and a collection of academic and personal projects.  

The goal of this project is to demonstrate mastery in:  
- Web structure using **Semantic HTML5**  
- Styling and layouts using **CSS3**  
- Deployment workflows using **Git & GitHub Pages**  

The website is fully responsive and provides a visually appealing experience across mobile, tablet, and desktop devices.

---

## Technical Requirements

### A. Structure (HTML5)
- Uses semantic HTML5 tags such as `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`, and `<article>`.
- Code is properly indented and commented.
- Document structure is logical, accessible, and organized.

### B. Styling
- Styling is done using **Custom CSS3**, including:
  - **CSS Variables (`:root`)** for consistent theme colors.
  - **CSS Grid** and **Flexbox** for layout.
  - Media queries for responsiveness.
  - Smooth hover effects and animations for UI elements.

### C. Responsiveness
- Fully responsive design that works on:
  - **Mobile devices (portrait)**
  - **Tablets**
  - **Desktop screens**
- Mobile navigation is optimized with a **hamburger menu**.

### D. Deployment
- Source code is hosted in a **public GitHub repository**.
- Live site is deployed via **GitHub Pages**.
- Repository includes this `README.md` to explain the project and tech stack.

## Content Features

### 1. Hero Section
- Introduction with my **name**, **tagline**, and a **Call to Action (CTA)** button linking to projects.  

### 2. About Me
- Professional biography
- Profile photo
- Brief overview of my background and skills

### 3. Skills
- Visual representation of technical skills using **progress bars, icons, and tags**.  
- Includes skills like HTML, CSS, JavaScript, Python, and Design.

### 4. Projects Gallery
- Showcases **3 main projects**:
  1. **Color Picker App**  
     - A simple app to pick colors and see hex/RGB values.  
     - Built with HTML, CSS & JS.
  2. **Mini To-Do App / Task Manager**  
     - Manage tasks with add, mark, and delete features.  
     - Built with HTML, CSS & JS.
  3. **Simple Calculator**  
     - Performs addition, subtraction, multiplication, and division.  
     - Built with HTML, CSS & JS.
